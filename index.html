PROJETO INICIAÇÃO CIENTÍFICA
TABELA PERIÓDICA TECNOLÓGICA ADAPTADA
SAIBA MAIS

OBJETIVO

Apresentar uma alternativa para se trabalhar com a Tabela Periódica através da construção de uma Tabela Periódica em Braille, utilizando impressão 3D e hardware de código fonte aberto "Arduino” e Promover a inclusão de alunos com deficiência visual nas redes de ensino regulares.

KIT DIDÁTICO

Contém uma tabela periódica em braille e 3 representações de átomos impressos em 3D, além de botões que quando acionados reproduzem audioaulas

SITE

Programado em HTML, CSS e javascript. A partir de uma "Lattees Template" produzida com bootstrap

Fotografias

Imagens fotografadas no SESI 403, onde o projeto foi pensado e produzido

Tecnologia

Foi utlizado impressões 3D para confecção dos átomos e da tabela e a plataforma de desenvolvimento Arduino junto a um módulo chamado WTV020-SD

AUDIOAULAS

Você pode fazer o download de novas audioaulas aqui, basta salvá-las no cartão SD sem alterar o nome do arquivo.

Você também pode adicionar suas próprias audioaulas desde que estejam no formato ".ad4" e possuam os nomes "0000,0001,0002...", sendo que acompanham a ordem dos botões do kit.
