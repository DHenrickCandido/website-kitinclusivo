SOBRE

Projeto realizado por alunos do 3°Ano do ensino médio, visando suprir a necessidade de material didático específico para portadores de deficiência visual, completa ou parcial, utilizando impressões 3D e microcontrolador.

KIT DIDÁTICO

O kit possui botões que quando apertados reproduzem audioaulas relacionadas à tabela.

SITE

Programado em HTML, CSS, e Javascript a partir de uma template Lattes.

Fotos

Fotografadas na escola SESI - 403, local onde o projeto teve início.

TECNLOGIA

Microcontrolador Arduino unido ao módulo, para cartão SD, wtv020-SD
